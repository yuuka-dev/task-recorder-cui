# CI セットアップ (Phase 7) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** push/PR で ruff (lint + format) と pytest + coverage を自動実行し、Codecov に Repository Token 方式でアップロードして README にバッジを貼る。

**Architecture:** GitHub Actions の単一ジョブ (`test`) を Python 3.11 / 3.12 / 3.13 のマトリクスで走らせ、Codecov はマトリクスの 3.11 かつ `push` イベント時のみ Repository Token 認証で coverage.xml を送る。pyproject.toml に `pytest-cov` と branch coverage 設定を追加し、README には CI / Codecov バッジとテスト節を加える。

**Tech Stack:** GitHub Actions, `actions/checkout@v4`, `actions/setup-python@v5`, `codecov/codecov-action@v5`, ruff, pytest, pytest-cov.

**関連ドキュメント:**
- 設計書: `docs/superpowers/specs/2026-04-14-ci-setup-design.md`
- プロジェクト仕様: `CLAUDE.md` (Phase 7 / ブランチ戦略 / CI バッジ)

## Task 3: `.github/workflows/ci.yml` を追加

**Files:**
- Create: `.github/workflows/ci.yml`

- [ ] **Step 1: `.github/workflows/` ディレクトリを作成**

Run: `mkdir -p .github/workflows`
Expected: 無出力で正常終了。

- [ ] **Step 2: `ci.yml` を以下の内容で作成**

ファイル: `.github/workflows/ci.yml`

```yaml
name: CI

on:
  push:
    branches: [main, dev]
  pull_request:
    branches: [main, dev]

concurrency:
  group: ci-${{ github.ref }}
  cancel-in-progress: true

jobs:
  test:
    runs-on: ubuntu-latest
    permissions:
      contents: read
    strategy:
      fail-fast: false
      matrix:
        python-version: ["3.11", "3.12", "3.13"]
    steps:
      - name: Checkout
        uses: actions/checkout@v4

      - name: Set up Python ${{ matrix.python-version }}
        uses: actions/setup-python@v5
        with:
          python-version: ${{ matrix.python-version }}
          cache: pip
          cache-dependency-path: pyproject.toml

      - name: Install project (dev extras)
        run: pip install -e ".[dev]"

      - name: Lint (ruff check)
        run: ruff check .

      - name: Format check (ruff format --check)
        run: ruff format --check .

      - name: Test with coverage
        run: pytest --cov=task_recorder_cui --cov-report=xml --cov-report=term

      - name: Upload coverage to Codecov
        if: matrix.python-version == '3.11'
        uses: codecov/codecov-action@v5
        with:
          token: ${{ secrets.CODECOV_TOKEN }}
          files: ./coverage.xml
          fail_ci_if_error: false
```

- [ ] **Step 3: YAML が構文的に妥当か確認 (yamllint または python 標準ライブラリで確認)**

Run: `python -c "import ast; open('.github/workflows/ci.yml').read()" 2>/dev/null || true; grep -c 'on:' .github/workflows/ci.yml > /dev/null && echo OK`
Expected: `OK`。またはローカルに `yamllint` がある場合は `yamllint .github/workflows/ci.yml` を使う。
Note: PyYAML は dev deps に含まれないため `import yaml` ではなく `yamllint` CLI か目視確認で代替する。

- [ ] **Step 4: コミット**

```bash

git commit -m "ci: GitHub Actions ワークフロー (lint/format/test/coverage) を追加"
```

Expected: コミットが作成される。
